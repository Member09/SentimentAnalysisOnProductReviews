    Masterworks

    Aurigo Masterworks is a cloud based enterprise class software suite that helps infrastructure owners plan and deliver large capital construction projects. The software suite automates every phase of the capital project construction lifecycle from capital planning, bidding, contract management, mobile inspection, right of way management, materials management, permitting, civil rights and asset maintenance. The entire platform is delivered over the cloud on the Amazon Web Services (AWS) cloud infrastructure and meets the highest levels of security, and is SSAE-16 Type II certified.

    PROCORE

    Procore is an online construction software for project management, cost management, communications and collaboration. Procore’s project management software includes features such as meeting minutes, drawing markups and document storage for all project-related materials. Procore's offerings also include an App Marketplace for partner integrations and custom third-party applications. Partners include enterprise file storage and content management company Box, Botlink, and Dexter + Chaney, an ERP provider.
